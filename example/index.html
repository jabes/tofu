Tofu.js

Originally developed in 2010 for use in Pinup, a graphical tool for tagging content within images on the web.

This is a very tiny JavaScript library that focuses on element manipulation including events, animations, and html parsing.

A git repo is available here.

Methods

addClass

Add one event listener to a provided element object that invokes a callback function.

Animate one or more CSS properties for a provided element object.
appendText

arrayUnique

Depending on the parameters provided, this method will add, remove, or get attribute values for a provided element object.
